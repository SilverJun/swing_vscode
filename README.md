# SWING Language VSCode Extension

SWING언어의 디버깅과 자동완성, 컴파일 등의 기능을 위해 구현한 Visual Studio Code의 Extension입니다.

## Features
* Support Debugging with VSCode.
* Support Snippet.
* Support Code Highlighting.
* Support Compile Command.
